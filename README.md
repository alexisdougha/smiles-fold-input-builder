# smiles-fold-input-builder
A repository to prepare input files for modified peptide structure prediction.

Generate an input file for [AlphaFold3](https://github.com/google-deepmind/alphafold3), [Boltz](https://github.com/jwohlwend/boltz) or [Chai](https://github.com/chaidiscovery/chai-lab) starting from the SMILES of any peptide generated by [**PEP-EDIT**](https://pep-edit.rpbs.univ-paris-diderot.fr/).
To see examples, simply run ```bash generate_examples.sh```.

The structures generated (and the code used on a HPC) with these inputs can be found in ```data/HPC```.
